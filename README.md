# Manual digital Better · INVEX

Sitio estático listo para GitHub Pages.

## Publicarlo

1. Sube **todo el contenido de esta carpeta** a la raíz de tu repositorio.
2. En GitHub abre **Settings → Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**.
4. Elige la rama `main`, carpeta `/ (root)` y presiona **Save**.

Los botones **Activación por APP** y **Activación por MAAT** llevan directamente a las diapositivas 34 y 51. Todos los archivos deben subirse juntos a la carpeta principal del repositorio; las imágenes `slide-01.webp` a `slide-59.webp` y `guia-errores-maat.pdf` quedan junto a `index.html`.
