# Manual digital Better · INVEX

Sitio estático listo para GitHub Pages.

## Publicarlo

1. Sube **todo el contenido de esta carpeta** a la raíz de tu repositorio.
2. En GitHub abre **Settings → Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**.
4. Elige la rama `main`, carpeta `/ (root)` y presiona **Save**.

Los botones **Activación por APP** y **Activación por MAAT** llevan directamente a las diapositivas 34 y 51. La guía de errores está incluida dentro de `assets/downloads/`.
